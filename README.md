# AI Exam Malpractice Detection System

I built this system to detect cheating during exams due to there being cbt exams

## Why I built this
Exam malpractice is a real issue in my environment due to the new computer based exams.  
I wanted to create a system that improves fairness.

## What it does
- Detects multiple faces
- Tracks head movement
- Monitors eye behavior
- Detects suspicious device movement
- Detects possible phone usage
- Uses a scoring system
- Terminates exam at high risk

## Tech used
- Python
- OpenCV
- MediaPipe

## Run
pip install -r requirements.txt  
python main.py
